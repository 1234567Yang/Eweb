# Eweb

## 用法

中文可视化HTML开发，可以直接点击组件拖放，表格编辑组件样式，并且可以实现无代码化前端设计。
支持了调整层次关系，可以实现前后顺序。
支持了组件自定位。

## 即将更新

位置默认为fixed，自动百分比化，并且支持添加组件使拖动切换大小。
